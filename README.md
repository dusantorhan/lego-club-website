# LEGO Robotics Club Website

This is my school project for creating a simple website.

The website is about a LEGO Robotics Club and includes several pages with basic functionality.

## How to run

1. Download or clone the project
2. Open the folder
3. Open index.html in a browser (Chrome or Edge)

For the projects page (JSON + AJAX), it is better to run the project using a local server.

## Pages

- Home
- About
- Activities
- Gallery
- Projects
- Contact

## What is included

- Simple website using HTML and CSS
- Navigation menu between pages
- Gallery with image preview (Lightbox)
- Contact form with JavaScript validation
- Projects table loaded from JSON using fetch (AJAX)

## Technologies used

- HTML
- CSS
- JavaScript
- JSON
